# 🌧️ Rainfall Prediction Project - UC9 (SENAC)

Rainfall Prediction Project developed as part of the UC9 (Curricular Unit 9) coursework at SENAC, focused on applying Machine Learning techniques to predict rainfall based on weather-related data.

## Project Description

This project aims to build a Machine Learning model capable of predicting rainfall using historical meteorological data.  
The model learns patterns from environmental and atmospheric variables to estimate the likelihood or occurrence of rain.

The project demonstrates the complete Data Science pipeline:
- Exploratory Data Analysis (EDA)
- Feature selection and engineering
- Model training and evaluation
- Performance analysis with metrics and visualizations

## Objectives

- Analyze weather data to identify patterns related to rainfall
- Train predictive models using Machine Learning algorithms
- Evaluate model performance using appropriate metrics
- Generate insights through data visualization

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Jupyter Notebook

# 🌧️ Projeto de Previsão de Chuva - UC9 (SENAC)

Projeto de Previsão de Chuva desenvolvido como atividade da Unidade Curricular 9 (UC9) do SENAC, com foco na aplicação de técnicas de Machine Learning para prever a ocorrência de chuva com base em dados meteorológicos.

## Descrição do Projeto

Este projeto tem como objetivo construir um modelo de Machine Learning capaz de prever a ocorrência de chuva utilizando dados meteorológicos históricos.  
O modelo aprende padrões a partir de variáveis ambientais e atmosféricas para estimar a probabilidade ou ocorrência de chuva.

O projeto demonstra todo o pipeline completo de Ciência de Dados:
- Análise Exploratória de Dados (EDA)
- Seleção e engenharia de variáveis (features)
- Treinamento e avaliação do modelo
- Análise de desempenho com métricas e visualizações

## Objetivos

- Analisar dados climáticos para identificar padrões relacionados à chuva
- Treinar modelos preditivos utilizando algoritmos de Machine Learning
- Avaliar o desempenho do modelo com métricas adequadas
- Gerar insights por meio de visualizações de dados  

## Tecnologias Utilizadas

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Jupyter Notebook
